# API REST com NodeJS

Para começar o projeto digite 
```bash 
$ npm init -y 
```

## Instalação de pacotes 

Gerenciador de rotas 
```
$  yarn add express
``` 

Para manipular json nas requisições 
```
$  yarn add body-parser 
``` 

Conexão com o banco de dados MongoDB
```
$ yarn add mongoose
```
